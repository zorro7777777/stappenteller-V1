Drie lagen van achtergrond-continuiteit

Audio Keepalive: speelt een onhoorbare 1s WAV-loop via Web Audio API. MediaSession API vertelt Android dat er media speelt. Chrome wordt behandeld als Spotify: hoge processprioriteit, zelden door Android gedood. Werkt zolang het scherm open is.

Crow-flies Re-sync: bij heropen berekent de app via Haversine de vogelvlucht-afstand tussen het punt van dichtklappen en het punt van heropen. Dit wordt toegevoegd als geschat segment (oranje ~GAP indicator). Nauwkeurig genoeg voor een stappenteller over korte slaapperiodes.

Zwarte Doos: elke 3 seconden schrijft de app een snapshot van de actieve sessie naar localStorage. Bij crash of gedwongen afsluiten door Android leest de app de snapshot bij herstart en hervat automatisch. Je verliest nooit meer dan 3 seconden data.

Eerlijke grens

Voor continue GPS-meting bij een dichtgeklapt scherm zonder onderbreking is een native Android-app met een Foreground Service de enige betrouwbare oplossing. De technieken hier zijn de best haalbare voor een browser-PWA.

Step Counter — Cockpitv2.0

Three layers of background continuity

Audio Keepalive: plays an inaudible 1s WAV loop via Web Audio API. The MediaSession API tells Android that media is playing. Chrome is treated like Spotify: high process priority, rarely killed. Works while the screen is open.

Crow-flies Re-sync: on re-open, the app calculates the straight-line distance between the closing and opening coordinates via Haversine, added as an estimated segment (orange ~GAP indicator). Accurate enough for a step counter over short sleep periods.

Black Box: every 3 seconds the app writes a session snapshot to localStorage. On crash or forced close by Android, the snapshot is read on restart and the session resumes automatically. You never lose more than 3 seconds of data.

Honest limitation

For continuous GPS tracking with screen closed without interruption, a native Android app with a Foreground Service is the only reliable solution. The techniques here represent the best achievable for a browser PWA.
